키워드 : 디지털 트랜스포메이션

경영학 메타가 변하는게 산업혁명 요인 중 하나
전략, 마케팅, 인사(HR), 생산

1차 산업혁명 : 기계가 일을 대신한다, 증기
장점 : 생산성, 시간적 효율, 고출력(대량생산)

마케팅
인사(HR) -> 교대근무
생산 -> 소품종 대량생산

2차 산업혁명 : 컨베이어 시스템 / 라인 프로세스, 3S 시스템(분업특화, 표준화, 단순화)
전략 -> 가격, 품질(6시그마) : 편차가 적어야 됨
마케팅 -> 불량률 적음
인사(HR)
생산 -> 소품종 대량생산, 라인 프로세스(한 라인에서 구분지어 생산)

3차 산업혁명 : 정보화 시대
전략 -> 맞춤형
마케팅 -> 프리미엄
인사(HR) -> 인적자원 개발, 직원 교육
생산 -> 다품종 소량생산, 배치 프로세스(라인별 장비설치) -> 반도체 공정

4차 산업혁명
기업에서는 아직까지는 변화에 보수적으로 접근하고 있음
블록체인, 딥러닝을 이용해서 어떻게 돈을 벌까??
'인간은 존엄하다'라는 가치관을 가장 먼저 내세우는 추세
개인의 특성을 모두 고려해주어야 한다
디지털 트랜스포메이션
	- 비즈니스
	- 기술 

기술을 모방하기 쉽다

초연결이 왜 힘들까?
1. 데이터 사일로(data silo) : 이해관계자들 간의 데이터 공유가 이루어지지 않는 현상
2. 분업의 문제 : 데이터 분석을 데이터 분석가가 전담하는 구조
	- 데이터 분석의 목적 설명 및 결과 해석은 해당 분야의 전문가(DE)가 해야 함
	- 데이터 분석가가 모든 시점에 대해 알지 못한다
3. 데이터 만능주의 : 데이터를 지나치게 신뢰하는 현상
	- 도덕성 등 고려해야할 사항들이 있음
4. 데이터 수집 전략의 부재
	- 로봇 청소기 카메라 : 집을 찍은 영상이 전송된다?? -> 법적 문제 고려하지 않고 데이터 모으려는 것이 문제
	- 맥락 수집 : 무엇을 어디까지 수집할 것인가
	- 고객 설득 : 어떻게 수집할 것인가, 데이터 수집에 대해 소비자들을 어떻게 설득할 것인가
	- 정보 보안 : 개인 정보 관리 어떻게 보장할 것인가
5. 기능 중심의 접근 : 데이터를 수단이 아니라 목적으로 보는 관점
	- LG -> 디바이스 별 구분
	- 서비스를 만들고 싶은데 어떤 데이터를 모아야 할까?라고 생각해야 함
	- 데이터가 있는데 이것으로 무엇을 할 수 있을까? (X)
	-> 목적을 가지고 필요한 데이터만 사용해야 한다.
6. 기술에 대한 이해의 부재
	- 사용하려고하는 기술에 대해서는 어느정도 이해를 하고 있어야 한다
	- 그래야 시간, 비용적인 고려를 할 수 있음
7. 표준화 문제 : 분야마다 다른 기준, 관점 등을 무시하는 문제
	- 개발 환경과 실제 환경은 많이 다름(나라마다, 조직마다 다름)
	- 기준점을 어느 쪽으로 잡았느냐에 따라 개발 환경이 다르다

직원 해야할 일이 늘어난다 -> 인센티브로 중재 : 협상을 해야됨
기술-비즈니스 모델 간 중개자가 존재해야 함


MaaS
모빌리티
써드파티 : 보험, 튜닝, 세차, 보안

모빌리티 + 써드파티 초연결 기술 => 영향력을 가질 수 있음. 독과점 문제

현대차
자동차 잔존가치 평가 : 엑셀 밟는 힘, 차량 관리 정도, 주행 속도, 외부 출격량
-> 안전운전지표 만들어서 보험사, 중고차 매장, 렌터카 매장에 영향
보험사 -> 안전운전지표에 따라 보험료 줄여줌, 사고비용 절약
중고차 -> 자동차 사기 빈도 감소.
렌터카 -> 난폭운전 줄여서 사고빈도 줄임. 렌터 비용 줄여줌.

디지털 생태계 전략
1. 네트워크 효과
	- 구성원이 많아지면 네트워크 크기가 기하급수적으로 늘어남
2. 플라이휠 모델 
	- 아마존이 1위
	- 낮은 가격으로 시장 진입
3. 락-인 전략
	- 애플이 1위
	- 고객 이탈 막는 전략, 구독 서비스
	- 신규 고객에게 기업을 알리고, 왜 선택해야하고
	1) 전환 비용 증가 전략
		기기 변경하면 충전기 등 액세서리 다 변경해야 됨
	2) 네트워크 활용
		기존 연결된 네트워크 사용 금지시켜서 안나가게 만듬

디지털 생태계 모델
	- 커플링 : AI회사-보컬트레이닝(예술적 감각), 뱅크 오브 와인(와인 소유권을 NFT로 관리)
	- 디커플링 : 배달의 민족(식당-배달-고객), 퍼플독

플랫폼 전략
먼저 들어가서 네트워크 형성해 놓은 쪽이 이긴다. 선점자 우위 필요
요즘은 후발주자 플랫폼 전략도 나오긴 함

전공과 코딩을 접목하는 사업

메타버스+ai+의학
ai로 임상실험 전 단계까지 시간을 줄이고 약을 개발할 때 임상실험 등 결과를 빨리 나오게 할 수 있음

블록체인+자동차+정비소
부품교체 등 거짓으로 할 수 없게 자동차 자체에 기술 적용
